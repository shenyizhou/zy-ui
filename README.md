## Build Setup

```bash
# install dependencies
npm install

# serve with hot reload at localhost:8080
npm run dev

# build for production with minification
npm run build
```

## Quick Start

```javascript
import Vue from 'vue';
import ZyUI from 'zy-ui';

Vue.use(ZyUI);

// or
import {
  InputGroup,
  // ...
} from 'zy-ui';

Vue.component(InputGroup.name, InputGroup);
```
